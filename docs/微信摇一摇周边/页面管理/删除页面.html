删除页面

接口说明

删除已有的页面，包括在摇一摇页面出现的主标题、副标题、图片和点击进去的超链接。只有页面与设备没有关联关系时，才可被删除。

接口调用请求说明

http请求方式: POST（请使用https协议）https://api.weixin.qq.com/shakearound/page/delete?access_token=ACCESS_TOKENPOST数据格式：json
POST数据例子：
{
   "page_id":34567
}

参数说明

参数是否必须说明
page_id 是指定页面的id   


返回说明

 正常时的返回JSON数据包示例：

{
   "data": {
       
   },
   "errcode": 0,
   "errmsg": "success."
}
