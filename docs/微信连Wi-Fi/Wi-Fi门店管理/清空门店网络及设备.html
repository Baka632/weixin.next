清空门店网络及设备


通过此接口清空门店的网络配置及所有设备，恢复空门店状态。

接口调用说明

协议：https
http请求方式: POST
请求URL：https://api.weixin.qq.com/bizwifi/shop/clean?access_token=ACCESS_TOKEN
POST数据格式：JSON

请求参数说明

参数是否必须说明
access_token是调用接口凭证
POST数据是JSON数据

POST数据

数据示例：

{
    "shop_id": 429620
}

字段说明：

字段是否必填说明
shop_id是门店ID
ssid否无线网络设备的ssid。若不填写ssid，默认为清空门店下所有设备；填写ssid则为清空该ssid下的所有设备


返回数据说明 正确返回JSON数据示例：

{
  "errcode": 0,
}
