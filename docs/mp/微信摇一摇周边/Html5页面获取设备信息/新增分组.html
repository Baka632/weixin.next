新增分组

接口说明

新建设备分组，每个帐号下最多只有1000个分组。

接口调用说明

http请求方式: POST（请使用https协议）https://api.weixin.qq.com/shakearound/device/group/add?access_token=ACCESS_TOKENPOST数据格式：json
POST数据例子：
{
  "group_name":"test"
}

参数说明

参数是否必须说明
access_token是调用接口凭证
group_name是分组名称，不超过100汉字或200个英文字母   


返回说明

正常时的返回JSON数据包示例：

{
  "data": {
      "group_id" : 123,
      "group_name" : "test"
  },
  "errcode": 0,
  "errmsg": "success."
}

参数说明

参数说明
group_id 分组唯一标识，全局唯一
group_name分组名
