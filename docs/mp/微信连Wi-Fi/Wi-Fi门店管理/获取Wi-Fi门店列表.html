获取Wi-Fi门店列表

 通过此接口获取WiFi的门店列表，该列表包括公众平台的门店信息、以及添加设备后的WiFi相关信息。创建门店方法请参考“微信门店接口”。

  注：微信连Wi-Fi下的所有接口中的shop_id，必需先通过此接口获取。

接口调用说明

协议：https
http请求方式: POST
请求URL：https://api.weixin.qq.com/bizwifi/shop/list?access_token=ACCESS_TOKENPOST
数据格式：JSON

请求参数说明

参数是否必须说明
access_token是调用接口凭证
POST数据是JSON数据

POST数据

数据示例：

{
  "pageindex": 1,		
  "pagesize":2
}

字段说明：

字段是否必填说明
pageindex否分页下标，默认从1开始
pagesize否每页的个数，默认10个，最大20个

返回数据说明 

正确返回JSON数据示例：

{
  "errcode": 0,
  "data": {
    "totalcount": 16,
    "pageindex": 1,
    "pagecount": 8,
    "records": [
      {
        "shop_id": 429620,
        "shop_name": "南山店",
        "ssid": "WX123",
        "ssid_list": [
          "WX123",
          "WX456"
        ],
        "protocol_type": 4,
        "sid": ""
      },
      {
        "shop_id": 7921527,
        "shop_name": "宝安店",
        "ssid": "",
        "ssid_list": [],
        "protocol_type": 0,
        "sid": ""
      }
    ]
  }
}

字段说明：

字段说明
totalcount总数
pageindex分页下标
pagecount分页页数
records当前页列表数组
shop_id门店ID
shop_name门店名称
ssid无线网络设备的ssid，未添加设备为空，多个ssid时显示第一个
ssid_list无线网络设备的ssid列表，返回数组格式
protocol_type门店内设备的设备类型，0-未添加设备，1-专业型设备，4-密码型设备，5-portal自助型设备，31-portal改造型设备
sid商户自己的id，与门店poi_id对应关系，建议在添加门店时候建立关联关系，具体请参考“微信门店接口”
